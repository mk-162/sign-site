ğŸ“‹ FILE FORMAT DETAILS

PNG (Portable Network Graphics)
  Resolution: 4x scale, 300 DPI
  Transparency: YES
  Best for: Web, presentations, digital use

TIFF (Tagged Image File Format)
  Resolution: 4x scale, 300 DPI
  Compression: LZW
  Transparency: YES
  Best for: Professional printing, archival

JPEG (Joint Photographic Experts Group)
  Resolution: 4x scale, 300 DPI
  Quality: 100%
  Transparency: NO (white background)
  Best for: Email, documents, general sharing

SVG (Scalable Vector Graphics)
  Type: Vector (scalable to any size)
  Transparency: YES
  Best for: Responsive web design, any size application

GIF (Graphics Interchange Format)
  Resolution: 4x scale
  Colors: 255 color palette
  Transparency: YES
  Best for: Web compatibility, simple graphics

.AF (Affinity Format)
  Type: Editable source files
  Layers: Fully preserved
  Software: Affinity Designer/Photo (free)
  Best for: Logo editing and customization
