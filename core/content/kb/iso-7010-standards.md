# ISO 7010 Safety Sign Standards Explained

ISO 7010 is the international standard for safety symbols. Understanding it is essential for workplace compliance.

## What is ISO 7010?

ISO 7010:2019 specifies safety signs used in workplaces and public facilities. It was designed to provide a globally consistent set of safety symbols that transcend language barriers.

## Sign Categories

### Prohibition Signs (Red Circle)

- White background, red border and diagonal line
- Black symbol
- Example: No Smoking, No Entry

### Warning Signs (Yellow Triangle)

- Yellow background, black border
- Black symbol
- Example: Electrical Hazard, Slippery Floor

### Mandatory Signs (Blue Circle)

- Blue background
- White symbol
- Example: Wear Eye Protection, Hard Hat Required

### Safe Condition Signs (Green Rectangle)

- Green background
- White symbol
- Example: Fire Exit, First Aid

### Fire Equipment Signs (Red Rectangle)

- Red background
- White symbol
- Example: Fire Extinguisher, Fire Alarm

## Key Requirements

1. **Size**: Must be proportional to viewing distance
2. **Colors**: Must match exact specifications (RAL colors)
3. **Symbols**: Must use approved pictograms only
4. **Placement**: Must be visible and unobstructed

## Viewing Distance Formula

```
Sign height = Viewing distance / 200
```

For example, a sign at 10m viewing distance needs a minimum height of 50mm.
